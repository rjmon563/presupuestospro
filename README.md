# PresupuestosPro

Aplicación PWA para autónomos y profesionales de reformas.

## Funciones
- Gestión de clientes
- Obras por cliente
- Mediciones y partes de trabajo
- Calculadora integrada
- Guardado local (offline)
- Instalación como app (PWA)

## Uso
1. Abrir `index.html` desde un servidor local
2. Instalar como PWA desde el navegador
3. Funciona offline

## Tecnologías
- HTML
- JavaScript
- Tailwind CSS
- PWA (Service Worker)

## Estado
Versión de pruebas para uso personal.
